Rundown: This assignment was challenging in many ways. Having not made the HTML code there was much to decypher and filter through in order to understand the website. That being said, once the program was largely understood, handling the styling side of the program introduced its own difficulties. These were things like using the right selectors, getting items side by side, positioning them in the right spot and making them responsive in order to work with differing viewports. These were eventually solved through continual testing, searching through external resources and reviewing class projects.

I learned alot from this assignment as it helped me understand where the issues arise while styling websites. This gives me an idea of what to think about when it comes to debugging my code in the future. I also learned a lot of new CSS terminology that should help expand my capabilities when styling. Along with new terminology, I also managed to really learn some of the basic code involved with CSS.

Previous class assignments seemed to be the most useful resource for completing the project. Having had many different topics in CSS at my disposal for review and help, it proved very useful when dealing with issues regarding those topics. While no specific website proved useful during this session, I did repeatedly obtain help from sources like Stack Overflow and wikiHow. These seemed to present relevant issues that other programmers had that helped me solve my own coding challenges. All in all though, I ended up receiving help from a number of different sources. I refferred to these sources particularly when it seemed as though I was missing a piece of documentation that might make things easier, and more efficient.
